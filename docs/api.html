WPStocks API

http:///[site]/api/? GET

stock
optional

Stock symbol to get information for. If not supplied then all stock for the current user if any will be retrieved.

username
optional

If supplied then will get stock information for the user matching username.

from
optional

If supplied then will get stock information for a stock from a certain date. If used then the stock parameter must not be empty. Defaults to current date.

to
optional

If supplied then will get stock information for a stock up to a certain date. If used then the stock parameter must not be empty. Defaults to current date.

frequency
optional

Can be "daily", "weekly" or "monthly". Defaults to "daily".

Examples

Get current stock information for stock symbol "APPLE":

http://[site]/api/?stock=APPLE

Get list of stocks that user "Bob" owns and return current information for each stock:

http://[site]/api/?username=Bob

Get weekly stock information for "APPLE", from 1 August 2015 to 1 September 2015:

http://[site]/api/?stock=APPLE&from=1 August 2015&to=1 September 2015&frequency=weekly
