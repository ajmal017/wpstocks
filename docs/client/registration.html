Creating your first capture

How to create a Capture

First, we will create our Capture.

  1. Step 1: Click on the 'Captures' tab and enter a name such as "First Capture."
  2. Step 2: In the "Event" dropdown, select "Visitor returns to page".
  3. Step 3: In the "Action" dropdown, select "Show message".
  4. Step 4: In the "Type" dropdown, select "Lightbox".
  5. Step 4: In the "Message" field, enter some text.
  6. Step 4: Click "Create New Capture". Be sure to copy the code that appears.

Next, we will create a web page that will use the Capture we just created.

  1. Step 1: Create a new html page. You can use the following as a template: 
                <!doctype html>
            <html>
               <head>
                  <meta charset="UTF-8">
                  <title></title>
               </head>
               <body>
               </body>
            </html>
  2. Step 2: Just before the closing body tag, add the Capture code that you saved.
  3. Step 3: Save your html page.

Next, testing our Capture.

  1. Step 1: Point your browser to the html page you created above. At this stage you should just see a blank page.
  2. Step 2: Refresh the page. You will see a lightbox appear with the text you entered in the message field.
