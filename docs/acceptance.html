Control Panel

Control panel tests
Test Date Accepted (Y/N)
"Dashboard" tab is selected by default.
On the default page user is shown a clients detail section and an accounts section.
In the client details section is shown the clients' name, company, address, telephone and fax number.
In the accounts section is shown the client's accounts
User is able to add a new account by clicking on the Add button in the accounts section.
Each line in the accounts table shows the correct fields.
Clicking on the portfolio tab shows the user's portfolio.
Portfolio table shows correct fields and balance.
Clicking on a stock symbol brings up a window showing graph showing the performance of the stock. User is able to change the graph to show daily, weekly and monthly results.
Underneath the statement of account table there is a watchlist table showing the stocks the user is watching.
User can add a stock to his watchlist by clicking on the "Add watch" button and entering a stock symbol
User can remove a stock from his watchlist by clicking the "Remove" button for that stock.
